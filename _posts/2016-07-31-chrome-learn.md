### 快捷操作
```
Cmd + Opt + J          //打开控制台
Cmd + Shift + C        //在审查模式下打开控制台
Cmd + [ / Cmd + ]      //控制台tab快速切换
Cmd+Shift+N            //新建隐身窗口
```

#### Sources
```
Cmd + O                //快速索引文件
Cmd + Shitf + O        //快速索引函数/特定选择器
Cmd + Opt + F          //在所有文件中搜索文字
```
